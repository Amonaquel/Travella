# Travella
?? Travella
Travella is a modern travel planning web application that helps users discover destinations, build itineraries, and manage trips seamlessly.
?? Features
* Destination discovery with detailed info
* Itinerary builder with custom plans
* User registration and login
* Interactive map integration
* Fully responsive design
??? Tech Stack
Frontend: React, Vite, CSS3
Backend: Node.js, Express.js, MongoDB
?? Project Structure
* travella-frontend/ — React-based frontend
* travella-backend/ — Express-based backend
* Future Plans.txt — Notes on upcoming features
* README.md — Project documentation
?? Getting Started
Prerequisites
* Node.js installed
* MongoDB running (local or Atlas)
Installation Steps
1. Clone the repository:
git clone https://github.com/Amonaquel/Travella.git
cd Travella
2. Install frontend dependencies:
cd travella-frontend
npm install
3. Install backend dependencies:
cd ../travella-backend
npm install
4. Create a .env file inside travella-backend with the following content:
ini
CopyEdit
PORT=5000
MONGODB_URI=your_mongodb_connection_string
5. Run the backend:
npm start (inside travella-backend)
6. Run the frontend:
cd ../travella-frontend
npm run dev
7. Open your browser at: http://localhost:3000
